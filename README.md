# 你画我猜网页游戏

这是一个 Node.js + Express + Socket.IO 写的网页版“你画我猜”。

## 功能

- 手机和电脑浏览器都能打开
- 输入昵称
- 输入房间号加入，房间不存在会自动创建
- 最多 6 名玩家
- 第一个加入的是房主
- 玩家准备 / 已就绪反复切换
- 所有人已就绪后，房主进入轮数设置
- 按房间玩家顺序轮流当画者
- 画者看到词条和提示，猜者只看到提示
- 绘画时间 90 秒
- 超时后显示答案，并进入 60 秒讨论时间
- 猜者答对：猜者 +2 分，画者 +1 分，显示 4 秒后进入下一位画者
- 支持画笔、色板、橡皮、撤回

## 本地运行

在 VS Code 中打开本文件夹，然后打开终端执行：

```bash
npm install
npm start
```

浏览器打开：

```text
http://localhost:3000
```

同一局域网手机测试：

1. 电脑和手机连接同一个 Wi-Fi
2. 电脑终端输入：

```bash
ipconfig
```

macOS / Linux 用：

```bash
ifconfig
```

3. 找到电脑局域网 IP，例如 `192.168.1.8`
4. 手机浏览器打开：

```text
http://192.168.1.8:3000
```

## 部署到公网

要让任意陌生人都能点开网址，需要部署到支持 Node.js 和 WebSocket 的平台，例如 Railway、Render、云服务器等。

常见流程：

1. 把本项目上传到 GitHub
2. 在 Railway 或 Render 创建 Node.js Web Service
3. 连接 GitHub 仓库
4. Start Command 填：

```bash
npm start
```

5. 部署完成后，平台会给你一个公网网址，别人打开这个网址就能玩。
